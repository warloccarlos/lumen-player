# 💎 Lumen | Modern Privacy-Focused Media Player

**Lumen** is a high-performance, glassmorphic web-based media player designed for the modern era. Built with a "Privacy-First" philosophy, Lumen allows you to play local files and remote streams directly in your browser without ever uploading your data to a server.



## 🛡️ Privacy & Security
Unlike traditional streaming platforms or "web players," Lumen operates on a **Zero-Knowledge** architecture:
* **Local Processing:** Local files are handled via the Browser File API. Your videos are never uploaded, buffered, or "seen" by any server.
* **No Tracking:** Lumen does not use cookies, analytics, or trackers. 
* **No Accounts:** No login is required. Your media habits remain your business.
* **HTTPS Only:** When hosted on GitHub Pages, all traffic is encrypted, ensuring your remote stream URLs remain private.

## ✨ Key Features
* **Hybrid Playback:** Seamlessly switch between local files (MP4, MKV, WebM) and remote URLs (HLS, DASH, MP4).
* **PWA Ready:** Install Lumen on your desktop or mobile device for a standalone, app-like experience.
* **Modern UI:** A sleek "Glassmorphism" interface designed for 2026, featuring a toggleable playlist and theater mode.
* **Drag & Drop:** Simply drop files onto the player to start watching instantly.
* **Auto-Advance:** Your playlist automatically moves to the next track for a continuous viewing experience.

## 🚀 Quick Start
1.  **Clone the Repo:** `git clone https://github.com/warloccarlos/lumen-player.git`
2.  **Open:** Launch `index.html` in any modern evergreen browser (Chrome, Edge, Firefox, Brave).
3.  **Install:** Click the "Install" icon in your browser's address bar to use Lumen as a standalone app.
